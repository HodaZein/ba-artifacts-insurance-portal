# RACI

R = responsible, A = accountable, C = consulted, I = informed.

- scope definition - BA: R, PO: A, tech: C, sponsor: C, compliance: C, QA: I
- requirements elicitation - BA: R/A, PO: C, tech: C, sponsor: I, compliance: C, QA: I
- process modelling (BPMN) - BA: R/A, PO: C, tech: C, compliance: C
- user stories + AC - BA: R, PO: A, tech: C, compliance: C, QA: C
- backlog prioritisation - BA: C, PO: R/A, tech: C, sponsor: C
- solution design - BA: C, PO: C, tech: R/A, compliance: C
- test strategy + UAT plan - BA: R, PO: A, tech: C, compliance: C, QA: R
- go / no-go - BA: C, PO: C, tech: C, sponsor: R/A, compliance: C, QA: C
- post-launch metrics review - BA: R, PO: A, tech: C, sponsor: C, QA: C
